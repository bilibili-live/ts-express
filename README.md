Github: https://github.com/lison16


## setup

```bash
# 请安装 `ts-node-dev`
cnpm i -g ts-node-dev
git clone https://github.com/bilibili-live/ts-express ts
cd ts
cnpm i
yarn start:dev
```

将 `.env.example` 改成 `.env` 自行配置 😊
